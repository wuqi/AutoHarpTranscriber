# 简谱规范 (jianpu-spec)

> 本规范基于 [番茄简谱脚本规范 v1.0](https://github.com/yetist/jianpu-spec)，描述本项目生成的简谱格式。

## 一、描述头（元数据）

描述头位于文件顶部，每行一个字段，格式为 `大写字母: 值`。

| 字段 | 含义 | 必填 | 示例 |
|:---|:---|:---|:---|
| `V` | 脚本版本号 | 是 | `V: 1.0` |
| `B` | 曲目标题 | 是 | `B: 小星星` |
| `D` | 调号（1=什么音） | 是 | `D: C` / `D: C#` / `D: C$` |
| `P` | 拍号 | 是 | `P: 4/4` / `P: 6/8` |
| `J` | 速度 (BPM) | 否 | `J: 120` |
| `Z` | 作者/编曲 | 否 | `Z: 传统民歌` |
| `X` | 序号 | 否 | `X: 1` |

### 调号规则

- 调号使用大写字母，升号用 `#`，降号用 `$`
- 例：`D: C`（C大调）、`D: F#`（F#大调）、`D: B$`（Bb大调）
- 本项目对关系小调自动转换为关系大调（如 Am → `D: C`）

### 注释

以 `#` 开头的行被视为注释，解析时忽略。

## 二、曲谱主体

### 2.1 行类型

| 前缀 | 含义 |
|:---|:---|
| `Q:` | 旋律行（曲） |
| `C:` | 歌词行（词），依附于上一行旋律 |
| `Q1:` / `Q2:` | 多声部旋律行（标注声部编号） |

### 2.2 基本音符

| 符号 | 含义 |
|:---|:---|
| `1` `2` `3` `4` `5` `6` `7` | 唱名 do re mi fa so la si（首调） |
| `0` | 休止符（可见） |
| `8` | 隐藏休止符（不可见但占位） |
| `9` | 节奏音符（无固定音高） |

### 2.3 八度标记

八度标记放在音符数字**之后**，多个可叠加。

| 符号 | 含义 | 示例 |
|:---|:---|:---|
| `'` | 升高一个八度（高音） | `1'` = 高音 do |
| `''` | 升高两个八度 | `1''` |
| `,` | 降低一个八度（低音） | `5,` = 低音 so |
| `,,` | 降低两个八度 | `5,,` |

> 本项目以 tonic 音（调式主音）在中音区八度 4 为基准，上方加 `'`，下方加 `,`。

### 2.4 变音记号

变音记号放在音符数字**之后**、八度标记**之前**。

| 符号 | 含义 | 示例 |
|:---|:---|:---|
| `#` | 升半音 | `4#` = fa 升半音 |
| `$` | 降半音（仅用于 D 字段调号） | `D: B$` |
| `=` | 还原号 | `4=` = fa 还原 |

> 与 JE 谱的重要区别：JE 谱将变音记号放在音符**之前**（如 `#4`），本规范放在**之后**。
>
> **MIDI→简谱转换说明**：MIDI 音符为绝对音高，升降一律使用 `#`，不使用 `$`。例如 tritone（三全音）记为 `4#` 而非 `4$`（`$` 仅在 D 字段调号中表示降号，如 `D: B$` = Bb 大调）。此约定与番茄简谱参考实现 (FQ.mid2fanqie) 一致。

### 2.5 时值记号

所有时值记号放在音符（含变音和八度标记）**之后**。

#### 增时线

| 符号 | 时值（以四分音符为 1 拍计） | 示例 |
|:---|:---|:---|
| （无后缀） | 四分音符 = 1 拍 | `5` |
| `-` | 二分音符 = 2 拍 | `5-` |
| `--` | 附点二分 = 3 拍 | `5--` |
| `---` | 全音符 = 4 拍 | `5---` |

#### 减时线

| 符号 | 时值 | 示例 |
|:---|:---|:---|
| `/` | 八分音符 = 0.5 拍 | `3/` |
| `//` | 十六分音符 = 0.25 拍 | `3//` |

#### 附点

| 符号 | 时值 | 示例 |
|:---|:---|:---|
| `.` | 附点（延长一半） | `5.` = 1.5 拍 |
| `/.` | 附点八分 = 0.75 拍 | `3/.` |

### 2.6 小节线

| 符号 | 含义 |
|:---|:---|
| `\|` | 标准小节线 |
| `\|/` | 隐藏小节线（不可见无占位） |
| `\|*` | 隐藏小节线（不可见但占位） |

### 2.7 其他符号

| 符号 | 含义 |
|:---|:---|
| `( ... )` | 连音线 / 圆滑线 |
| `(y ... )` | 多连音（软件自动计数） |
| `[ ... ]` | 前倚音；括号内支持八度和减时线 |
| `[h ... ]` | 后倚音 |
| `&zkh` / `&ykh` | 左/右伴奏括号 |
| `{bz ... }` | 伴奏声部 |
| `{dsb ... }` | 临时多声部 |
| `[fenye]` | 手动分页 |

> 注意：本项目 v1.0 暂不支持倚音、伴奏、多声部等高级功能。

## 三、完整示例

### 示例 1：《小星星》C大调

```
V: 1.0
B: 小星星
D: C
P: 4/4
J: 96
Q: | 1 1 5 5 | 6 6 5- | 4 4 3 3 | 2 2 1- |
Q: | 5 5 4 4 | 3 3 2- | 5 5 4 4 | 3 3 2- |
Q: | 1 1 5 5 | 6 6 5- | 4 4 3 3 | 2 2 1- |
```

### 示例 2：含八度标记和变音

```
V: 1.0
B: 示例曲
D: C
P: 4/4
J: 100
Q: | 1' 7 6 5 | 4# 5 6, 1 | 3/ 2/ 1 7, | 5,- - - |
```

说明：
- `1'` = 高音 do
- `4#` = fa 升半音
- `6,` = 低音 la
- `5,-` = 低音 so 二分音符
- `3/ 2/` = mi re 各八分音符

## 四、与 JE 谱 (je_score_operator) 的关键差异

| 特性 | 本规范 (jianpu-spec) | JE 谱 |
|:---|:---|:---|
| 高八度 | `'` (音符后) | `[]` 方括号 |
| 低八度 | `,` (音符后) | `()` 圆括号 |
| 升号位置 | 音符后 `4#` | 音符前 `#4` |
| 降号 | `$` | `b` |
| 节奏信息 | 支持（增时线/减时线/附点） | 不支持 |
| 休止符 | `0` | 空格/换行 |

## 五、本项目实现说明

### 已实现

- 描述头：V, B, D, P, J
- 基本音符 1-7 + 休止符 0
- 八度标记 `'` `,`
- 变音记号 `#` `$`
- 增时线 `-`、减时线 `/`、附点 `.`
- 小节线 `|`
- 首调 movable-do 体系

### 未实现 (v1.0 范围外)

- 还原号 `=`（MIDI→简谱管线中不涉及临时记号取消）
- 连音线 `()`、倚音 `[]`、多连音 `(y)`
- 多声部 `Q1` `Q2`
- 歌词行 `C:`
- 伴奏声部
- 手动分页
